#Expense Tracker (Python)

A console-based personal finance tracker built in Python.  
This project helps manage income and expenses while practicing Python OOP, file handling, and data persistence.

---

##Features

- Add income entries
- Add expense entries
- Display all transactions
- Calculate current balance
- Save data to JSON file

---

##Tech Stack

- Python
- Object-Oriented Programming (OOP)
- JSON File Handling
- Console application

---

##Project Structure


main.py → CLI interface
transaction.py → Transaction model
tracker.py → Core business logic
data.json → Saved transactions


---

##How to run

```bash
python main.py
